# Contributing

Thanks for helping grow **Awesome LangGraph Agents** — a curated list of AI agents built with [LangGraph](https://www.langchain.com/langgraph). Contributions of your own agents, or great ones you've found, are welcome.

## Adding an agent

1. Fork this repository and create a new branch.
2. Add your agent to the most relevant category in [`README.md`](README.md), following the existing format:

   ```markdown
   - **[Agent Name](https://link-to-your-agent)** — One concise line on what it does.
   ```

3. If no category fits, propose a new one in your pull request.
4. Open a pull request describing the agent.

## Guidelines

- The agent must be built with [LangGraph](https://www.langchain.com/langgraph).
- Link to a real, working resource — its docs, source repo, or a live demo.
- Keep the description to a single line, factual and free of marketing language.
- One agent per entry, and check it isn't already listed.
- Make sure every link works.
- Keep entries tidy and consistent with the surrounding list.

By contributing, you agree that your contributions are licensed under [CC0](LICENSE) and that you will follow our [Code of Conduct](CODE_OF_CONDUCT.md).
